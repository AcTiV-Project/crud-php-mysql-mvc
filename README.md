
# Employee Management System

Web application developed in PHP using the MVC architectural pattern, designed for employee management within a company.

##Main Features

- Employee registration with personal and employment data
- Employee search by ID number
- Employee information updates
- Record deletion
- Data validation in forms
- MVC architecture for better code organization and maintenance

##Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- MVC Pattern

##Technical Description

This project simulates a real system used by small and medium-sized businesses to manage their employee information.

The business logic is separated into models, controllers, and views, allowing for easy scalability and maintenance.

##Installation and Use

1. Clone the repository
2. Import the included MySQL database
3. Configure the database credentials
4. Run the project on a local server (XAMPP, WAMP, or similar)

##Project Objective

Demonstrate practical knowledge of backend development with PHP, relational database management, and application of the MVC pattern in a real-world environment.
